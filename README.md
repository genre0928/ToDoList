# ToDoList 제작
Recoil 라이브러리를 활용한 전역 상태 관리 및 effects_UNSTABLE을 통한 상태 변경 관리를 연습했으며,<br/>
git repository 생성부터 커밋까지 반복 연습을 통해 형상 관리를 연습했습니다.
<br/>
<br/>


## 프로젝트 개요
### 언어 및 라이브러리, 툴
언어 : Typescript<br/>
빌드 도구 : Vite<br/>
형상 관리 : Github<br/>
전역 상태 관리 : Recoil<br/>
CSS 라이브러리 : styled-components<br/>
